Ändert die Map Tiles und einige Elemente auf [verkehr.vz-deutschland.de](https://verkehr.vz-deutschland.de/) zu einer dunkleren Farbe.
 
## Installation
 
1. [Tampermonkey](https://www.tampermonkey.net/) installieren (Chrome, Firefox, Edge, Safari...)
2. [`verkehr-vz-darkmode.user.js`](./verkehr-vz-darkmode.user.js) öffnen oder manuell in Tampermonkey einfügen
 
## Nutzung
 
- Darkmode ist standardmäßig aktiviert
- Kann de-/aktiviert werden über das Tampermonkey-Icon (Menüpunkt variiert je nach Sprache).
